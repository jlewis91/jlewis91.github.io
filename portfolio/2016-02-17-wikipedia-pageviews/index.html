Wikipedia Data

Jan 1, 0001

At this point, just a note to myself about all of the myriad ways one can access Wikipedia data and what they are good for:

R Packages

Pageview Data

  • Pageviews: Wrapper for the latest official wikimedia pageview API. Offers the greatest detail including breakdown by device type (desktop/mobile) and traffic type (user/bot) for daily pageview data. Data available starting in Sept 2015.
  • wikipediatrend: Cool package that draws upon the legacy pageviews api, [stats.grok.se]() and offers data going back to Dec 2007. Doesn’t offer any segmentation, but does offer a function for querying all language versions of a page.

Wikidata

  • WikidataR: Wrapper for the wikidata api, allows access to tons of normed-data goodness.
  • Sparql Endpoint: Endpoint against which Sparql queries can be sent for executing more sophisticated wikidata inquiries. (Could wrap: https://github.com/cran/SPARQL)
